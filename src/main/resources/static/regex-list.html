Справочник регулярных выражений

Вернуться к проверке
Выражение Описание Пример
\ Экранирование спецсимвола Регулярка: a\.b
Текст: a.b a+b aab
Найдёт: a.b
\A Начало текста Регулярка: \Astart
Текст: start middle start
Найдёт: start (только первое)
\Z Конец текста Регулярка: end\Z
Текст: start end
Найдёт: end (только последнее)
\b Граница слова Регулярка: \bcat\b
Текст: cat concatenate my cat
Найдёт: cat, cat
\B Не граница слова Регулярка: \Bcat\B
Текст: concatenate cat
Найдёт: cat (в "concatenate")
\< Начало слова Регулярка: \<start
Текст: start restart starting
Найдёт: start, start
\> Конец слова Регулярка: end\>
Текст: end friend weekend
Найдёт: end, end
\d Цифра [0-9] Регулярка: \d\d
Текст: a12 b3 c456
Найдёт: 12, 45
\D Не цифра Регулярка: \D+
Текст: 123abc456
Найдёт: abc
\w Символ слова A-Z,a-z,0-9 Регулярка: \w+
Текст: hello_world test123!
Найдёт: hello_world, test123
\W Не символ слова Регулярка: \W+
Текст: hello!@#world
Найдёт: !@#
\s Пробельный символ Регулярка: \s+
Текст: hello world
Найдёт:
\S Не пробельный символ Регулярка: \S+
Текст: hello world
Найдёт: hello, world
^ Начало строки Регулярка: ^start
Текст: start line\nstart again
Найдёт: start (только в начале)
$ Конец строки Регулярка: end$
Текст: the endnot end
Найдёт: end (только в конце строки)
. Любой символ (кроме перевода строки) Регулярка: a.b
Текст: acb a b axb
Найдёт: acb, axb
() Группировка / подмаска Регулярка: (abc)+
Текст: abcabc xyz
Найдёт: abcabc
* 0 или более повторений Регулярка: a*b
Текст: b ab aab test
Найдёт: b, ab, aab
+ 1 или более повторений Регулярка: a+b
Текст: b ab aab test
Найдёт: ab, aab
? 0 или 1 повторение Регулярка: colou?r
Текст: color colour
Найдёт: color, colour
Литералы Обычные символы без спецзначения Регулярка: hello
Текст: hello world
Найдёт: hello
[0-9] Любая цифра от 0 до 9 Регулярка: [0-9][0-9]
Текст: 12 3 456
Найдёт: 12, 45
[abc] Любой символ из списка Регулярка: [abc]+
Текст: abc cab test
Найдёт: abc, cab
[^abc] Любой символ кроме указанных Регулярка: [^abc]+
Текст: abc def xyz
Найдёт: def xyz
[a-z] Любая строчная латинская буква Регулярка: [a-z]+
Текст: Hello world
Найдёт: ello, world
[A-Z] Любая заглавная латинская буква Регулярка: [A-Z][a-z]*
Текст: Hello World test
Найдёт: Hello, World
